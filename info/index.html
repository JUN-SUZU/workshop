サーバーアドレス

工房鯖は、工業化modのマルチプレイサーバーです。

サーバーに参加するには、Minecraft Java Editionが必要です。

サーバーアドレスは、以下の通りです。

サーバーアドレス: mc.jun-suzu.net

バージョン

工房鯖は、バージョン1.16.5に対応しています。

最新のバージョンでプレイすることをお勧めします。

mod

現在の工房鯖の主なmodは、MekanismとCreateとImmerive EngineeringとIndustrial Foregoingです。

modの詳細は、サーバー内の情報をご覧ください。
